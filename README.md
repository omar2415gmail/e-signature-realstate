# آثار العقارية — منصة توقيع العقود

موقع جاهز للنشر: المضيف يعبّي بيانات الإيجار ← يطلع رابط يرسله للعميل ← العميل يصادق ويوقّع ← يوصل العقد PDF لإيميل المؤجّر. **بدون قاعدة بيانات.**

الملفات:
- `index.html` — الموقع كامل (واجهة المضيف + العميل + توليد الـ PDF).
- `api/send.js` — دالة صغيرة ترسل العقد لإيميلك عبر Resend.

---

## الإعداد (مرة واحدة — حوالي ١٥ دقيقة)

### ١) احصل على مفتاح Resend (لإرسال الإيميل)
1. افتح **resend.com** وسجّل دخول **بإيميلك `omar2415@gmail.com`** (مهم: لازم نفس الإيميل اللي تبي تستقبل عليه العقود).
2. فعّل الإيميل من رسالة التحقق.
3. من القائمة: **API Keys ← Create API Key** ← انسخ المفتاح (يبدأ بـ `re_...`).

> ملاحظة: في الباقة المجانية، المرسِل `onboarding@resend.dev` يوصل لإيميل حسابك فقط — وهذا بالضبط اللي نبيه (تستقبل أنت). كافٍ تماماً لاستخدامك.

### ٢) ارفع الملفات على GitHub
1. افتح **github.com** ← New repository ← الاسم: `e-signature-realstate` ← Create.
2. اضغط **Add file ← Upload files** وارفع `index.html` ومجلّد `api` (داخله `send.js`). حافظ على بنية المجلّدات.
3. Commit.

### ٣) انشر على Vercel
1. افتح **vercel.com** وسجّل دخول بحساب GitHub.
2. **Add New… ← Project** ← اختر مستودع `e-signature-realstate` ← Import.
3. **Project Name:** اكتب `e-signature-realstate` (يصير الرابط `e-signature-realstate.vercel.app`).
4. افتح **Environment Variables** وأضِف:
   - **Name:** `RESEND_API_KEY`
   - **Value:** المفتاح اللي نسخته من Resend (`re_...`)
5. اضغط **Deploy**.

بعد دقيقة، موقعك شغّال على:
**https://e-signature-realstate.vercel.app**

---

## كيف تستخدمه يومياً
1. افتح الرابط من جوالك. أول مرة فقط: افتح قسم **"✍︎ توقيعي"** ووقّع — يُحفظ على جهازك ويظهر بكل عقد.
2. اختر الشقة، عبّي التواريخ وسعر الليلة والتأمين ← **إنشاء رابط العميل**.
3. اضغط **إرسال عبر واتساب** أو انسخ الرابط وأرسله للعميل.
4. العميل يفتح الرابط، يصادق، يعبّي بياناته، يوقّع، ويضغط **توقيع وإرسال**.
5. يوصلك العقد **PDF** على `omar2415@gmail.com`. احفظه وأرسل نسخة للعميل بالواتس.

## ملاحظات
- **الخصوصية:** الموقع لا يخزّن أي بيانات. بيانات الإيجار محمولة داخل الرابط، والعقد يُرسل لإيميلك ثم ينتهي.
- **حدود Resend المجاني:** ١٠٠ إيميل/يوم (٣٬٠٠٠/شهر) — أكثر من كفايتك.
- **تعديل الشقق أو الأسعار:** كلها في أعلى ملف `index.html` (ابحث عن `APTS` و `LESSOR`) — عدّلها وارفع التحديث على GitHub فيتحدّث الموقع تلقائياً.
- لو حبيت لاحقاً ترسل النسخة للعميل تلقائياً (لا يدوياً)، نضيف ربط واتساب الأعمال — قابل للتطوير.
